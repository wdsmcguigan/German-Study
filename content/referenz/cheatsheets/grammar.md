# Grammatik-Kurzübersichten

## Artikeldeklination: Alle 3 Typen

### Typ 1: Bestimmter Artikel (der/die/das)

| | maskulin | feminin | neutral | Plural |
|--|----------|---------|---------|--------|
| Nom. | der | die | das | die |
| Akk. | den | die | das | die |
| Dat. | dem | der | dem | den |
| Gen. | des | der | des | der |

**Adjektivendungen nach bestimmtem Artikel:**

| | m | f | n | Pl. |
|--|---|---|---|-----|
| Nom. | **-e** | **-e** | **-e** | **-en** |
| Akk. | **-en** | **-e** | **-e** | **-en** |
| Dat. | **-en** | **-en** | **-en** | **-en** |
| Gen. | **-en** | **-en** | **-en** | **-en** |

### Typ 2: Unbestimmter Artikel (ein/eine/kein)

| | maskulin | feminin | neutral | Plural |
|--|----------|---------|---------|--------|
| Nom. | ein/**-er** | eine/**-e** | ein/**-es** | keine/**-en** |
| Akk. | einen/**-en** | eine/**-e** | ein/**-es** | keine/**-en** |
| Dat. | einem/**-en** | einer/**-en** | einem/**-en** | keinen/**-en** |
| Gen. | eines/**-en** | einer/**-en** | eines/**-en** | keiner/**-en** |

### Typ 3: Ohne Artikel (starke Endungen)

| | m | f | n | Pl. |
|--|---|---|---|-----|
| Nom. | **-er** | **-e** | **-es** | **-e** |
| Akk. | **-en** | **-e** | **-es** | **-e** |
| Dat. | **-em** | **-er** | **-em** | **-en** |
| Gen. | **-en** | **-er** | **-en** | **-er** |

## Präpositionen: Kasus-Übersicht

### Akkusativ (always)
| Präp. | Beispiel |
|-------|---------|
| **für** | Ich kaufe ein Geschenk **für** dich. |
| **durch** | Wir gehen **durch** den Park. |
| **ohne** | Ich kann **ohne** dich nicht leben. |
| **gegen** | Er ist **gegen** die Idee. |
| **um** | Wir sitzen **um** den Tisch. |
| **bis** | Wir warten **bis** nächsten Montag. |
| **entlang** | Wir gehen den Fluss **entlang**. |

### Dativ (always)
| Präp. | Beispiel |
|-------|---------|
| **mit** | Ich gehe **mit** meinem Freund. |
| **von** | Das Geschenk ist **von** meinem Vater. |
| **zu** | Ich gehe **zur** Schule. |
| **bei** | Ich wohne **bei** meiner Tante. |
| **aus** | Ich komme **aus** der Türkei. |
| **nach** | Ich fahre **nach** Berlin. |
| **seit** | Ich lerne Deutsch **seit** einem Jahr. |
| **gegenüber** | Das Café liegt **gegenüber** dem Bahnhof. |
| **außer** | **Außer** mir kommt niemand. |

### Wechselpräpositionen (Dativ = Wo?, Akkusativ = Wohin?)
**in, an, auf, unter, über, vor, hinter, neben, zwischen**

## DA- und WO-Formen

| Frage (Sachen) | Antwort |
|----------------|---------|
| **Woran** denkst du? | Ich denke **daran**. |
| **Worauf** wartest du? | Ich warte **darauf**. |
| **Worüber** redest du? | Ich rede **darüber**. |
| **Wovon** träumst du? | Ich träume **davon**. |
| **Womit** fährst du? | Ich fahre **damit**. |

> ⚠️ Bei Personen: Präposition + wen/wem (nicht da-/wo-)
> - *Mit **wem** sprichst du?* (not: Womit?)

## Nebensatzkonjunktionen (Verb → Ende)

| Konjunktion | Bedeutung | Beispiel |
|-------------|-----------|---------|
| **weil** | because | Ich bleibe zu Hause, **weil** ich krank **bin**. |
| **dass** | that | Ich glaube, **dass** er kommt. |
| **wenn** | if / when | **Wenn** es regnet, bleiben wir hier. |
| **obwohl** | although | Er arbeitet, **obwohl** er müde **ist**. |
| **ob** | whether | Ich weiß nicht, **ob** sie kommt. |
| **damit** | so that | Ich lerne, **damit** ich bestehe. |
| **als** | when (past) | **Als** ich Kind war, spielte ich viel. |
| **seitdem** | since | **Seitdem** er hier wohnt, ist er glücklich. |
| **während** | while | Sie kocht, **während** er schläft. |
| **nachdem** | after | Er geht, **nachdem** er gegessen hat. |
| **bevor** | before | Sie trinkt Kaffee, **bevor** sie geht. |
| **bis** | until | Warte, **bis** ich fertig bin. |

## Verbindungsadverbien (Position 1 → Verb an Position 2)

| Adverb | Bedeutung | Beispiel |
|--------|-----------|---------|
| **deshalb / deswegen / darum** | therefore | Er ist krank, **deshalb** bleibt er zu Hause. |
| **trotzdem** | nevertheless | Es regnet. **Trotzdem** gehen wir. |
| **außerdem / zudem** | moreover | **Außerdem** ist es günstig. |
| **jedoch / allerdings** | however | **Jedoch** gibt es Probleme. |
| **dann / danach** | then / after that | Wir essen, **dann** gehen wir spazieren. |
| **anschließend** | subsequently | **Anschließend** trinken wir Kaffee. |
| **schließlich** | finally | **Schließlich** haben wir es geschafft. |

## Zweiteilige Konnektoren

| Paar | Bedeutung | Beispiel |
|------|-----------|---------|
| **nicht nur … sondern auch** | not only … but also | Er spricht **nicht nur** Deutsch, **sondern auch** Englisch. |
| **sowohl … als auch** | both … and | **Sowohl** Anna **als auch** Tim kommen. |
| **entweder … oder** | either … or | **Entweder** du kommst, **oder** ich gehe allein. |
| **weder … noch** | neither … nor | Er trinkt **weder** Kaffee **noch** Tee. |
| **zwar … aber** | although … but | Es ist **zwar** teuer, **aber** es lohnt sich. |

## Passiv (Passive Voice)

| Zeitform | Formel | Beispiel |
|---------|--------|---------|
| Präsens | wird + Partizip II | Das Paket **wird** transportiert. |
| Präteritum | wurde + Partizip II | Das Paket **wurde** transportiert. |
| Perfekt | ist + Partizip II + worden | Das Paket **ist** transportiert **worden**. |
| Futur | wird + Partizip II + werden | Das Paket **wird** transportiert **werden**. |

## Konjunktiv II: Übersicht

| Verb | ich | du | er/sie |
|------|-----|-----|--------|
| sein | wäre | wärst | wäre |
| haben | hätte | hättest | hätte |
| werden | würde | würdest | würde |
| können | könnte | könntest | könnte |
| müssen | müsste | müsstest | müsste |
| dürfen | dürfte | dürftest | dürfte |

**Verwendung:**
- Wunsch: Ich **würde gern** reisen. / Ich **wäre** lieber zu Hause.
- Vorschlag: Du **könntest** früher kommen.
- Höflichkeit: **Könnten** Sie mir helfen?
- Bedingung: **Wenn** ich Zeit **hätte**, würde ich kommen.

## Satzstellung (TEKAMOLO)

| Position | Typ | Frage | Beispiel |
|----------|-----|-------|---------|
| **TE** | Temporal | Wann? | heute, morgen, um 8 Uhr |
| **KA** | Kausal | Warum? | wegen der Arbeit, aus Liebe |
| **MO** | Modal | Wie? | gern, mit Freude, allein |
| **LO** | Lokal | Wo/Wohin? | in der Schule, nach Berlin |

> Normal order: Ich lerne Deutsch **heute** (TE) **wegen des Tests** (KA) **mit Freude** (MO) **in der Bibliothek** (LO).

## Indefinitpronomen: Positiv und Negativ

### Für Sachen
| Positiv | Negativ | Bedeutung |
|---------|---------|-----------|
| alles | nichts | everything / nothing |
| etwas | nichts | something / nothing |
| manches | keines | some / none |
| irgendetwas | nichts | anything / nothing |

### Für Personen
| Positiv | Negativ | Bedeutung |
|---------|---------|-----------|
| jemand | niemand | somebody / nobody |
| jeder | keiner / niemand | everyone / no one |
| irgendjemand | niemand | anybody / nobody |
