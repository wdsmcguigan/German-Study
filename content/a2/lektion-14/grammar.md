# Grammatik – Lektion 14: Lebensstationen

## Perfekt: Trennbare und Nicht-trennbare Verben (Revision)

### Trennbare Verben (ge- between prefix and stem)
| Infinitiv | Partizip II | hat / ist |
|-----------|-------------|-----------|
| kennen|lernen | **kennengelernt** | hat |
| auf|wachsen | **aufgewachsen** | ist |
| um|ziehen | **umgezogen** | ist |
| an|kommen | **angekommen** | ist |
| aus|wandern | **ausgewandert** | ist |
| auf|hören | **aufgehört** | hat |

### Nicht-trennbare Verben (no ge-)
| Infinitiv | Partizip II | hat / ist |
|-----------|-------------|-----------|
| bekommen | **bekommen** | hat |
| verstehen | **verstanden** | hat |
| erleben | **erlebt** | hat |
| vergessen | **vergessen** | hat |
| beginnen | **begonnen** | hat |
| gehören | **gehört** | hat |

### Verben auf -ieren (no ge-)
| Infinitiv | Partizip II | hat / ist |
|-----------|-------------|-----------|
| fotografieren | **fotografiert** | hat |
| heiraten | **geheiratet** | hat |
| studieren | **studiert** | hat |
| telefonieren | **telefoniert** | hat |

## Präteritum: sein und haben (Revision)

| Person | war (sein) | hatte (haben) |
|--------|-----------|--------------|
| ich | **war** | **hatte** |
| du | **warst** | **hattest** |
| er/sie/es | **war** | **hatte** |
| wir | **waren** | **hatten** |
| ihr | **wart** | **hattet** |
| sie/Sie | **waren** | **hatten** |

**Beispiele:**
- Als ich jung **war**, lebte ich in einem Dorf.
- Wir **hatten** ein Haustier.
- Es **war** eine schöne Zeit.

## Konjunktiv II: Wünsche über die Vergangenheit / Zukunft

| Ausdruck | Beispiel |
|---------|---------|
| Ich **wäre gern** ... | Ich wäre gern Ärztin geworden. |
| Ich **hätte gern** ... | Ich hätte gern mehr gelernt. |
| Wenn ich … **gewesen wäre**, … | Wenn ich reicher gewesen wäre, hätte ich studiert. |

## Satzverbindungen: weil – wenn – dass – ob

| Konjunktion | Bedeutung | Verb |
|-------------|-----------|------|
| **weil** | because | → Ende |
| **wenn** | when / if | → Ende |
| **dass** | that | → Ende |
| **ob** | whether | → Ende |

**Beispiele:**
- Ich bin glücklich, **weil** ich eine Familie **habe**.
- **Wenn** ich alt bin, möchte ich reisen.
- Ich denke, **dass** die Zukunft wichtig **ist**.
- Ich frage mich, **ob** ich richtig entschieden **habe**.

## Wortbildung: -ung Nomen (Noun Suffixes)

| Verb | → Nomen auf -ung |
|------|-----------------|
| entscheiden | die Entscheidung |
| erfahren | die Erfahrung |
| erinnerung | die Erinnerung |
| erfüllen | die Erfüllung |
| beschäftigen | die Beschäftigung |

> Verbs → nouns: add **-ung** to the verb stem. Always **feminin (die)**.

## immer + Komparativ (ever more / increasingly)

| Struktur | Beispiel |
|---------|---------|
| immer + Komparativ | Es wird **immer** besser. |
| | Die Welt wird **immer** kleiner. |
| | Er wird **immer** älter. |
