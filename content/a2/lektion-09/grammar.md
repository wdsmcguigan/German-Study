# Grammatik – Lektion 9: Meine Sachen

## Adjektivdeklination: Indefiniter Artikel (ein/eine/kein)

| | maskulin | feminin | neutral | Plural |
|--|----------|---------|---------|--------|
| Nom. | ein **neuer** Laden | eine **schöne** Wohnung | ein **neues** Licht | keine **braunen** Möbel |
| Akk. | einen **neuen** Laden | eine **schöne** Wohnung | ein **neues** Licht | keine **braunen** Möbel |
| Dat. | einem **neuen** Laden | einer **schönen** Wohnung | einem **neuen** Licht | keinen **braunen** Möbeln |

**Endungen Übersicht:**

| | m | f | n | Pl. |
|--|---|---|---|-----|
| Nom. | **-er** | **-e** | **-es** | **-en** |
| Akk. | **-en** | **-e** | **-es** | **-en** |
| Dat. | **-en** | **-en** | **-en** | **-en** |

**Beispiele:**
- Das ist ein **praktischer** Rucksack. (m, Nom.)
- Ich kaufe einen **neuen** Hut. (m, Akk.)
- Sie trägt eine **bunte** Jacke. (f, Akk.)
- Er wohnt in einem **kleinen** Zimmer. (n, Dat.)

## Vergleichspartikeln: als und wie

| Partikel | Bedeutung | Beispiel |
|---------|-----------|---------|
| **so … wie** | as … as (equal) | Das Buch ist **so** interessant **wie** der Film. |
| **-er als** | more … than (unequal) | Das Kleid ist teuer**er als** die Hose. |

**Weitere Beispiele:**
- Er läuft **schneller als** ich.
- Das neue Handy ist **genauso gut wie** das alte.
- Dieser Roman ist **interessanter als** jener.
