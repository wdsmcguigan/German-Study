# Grammatik – Lektion 8: Am Wochenende

## Konjunktiv II: Konjugation

### würde + Infinitiv (most verbs)

| Person | würde |
|--------|-------|
| ich | **würde** |
| du | **würdest** |
| er/sie/es | **würde** |
| wir | **würden** |
| ihr | **würdet** |
| sie/Sie | **würden** |

**Beispiele:**
- Ich **würde gern** ins Kino gehen.
- Er **würde lieber** zu Hause bleiben.
- Wir **würden am liebsten** reisen.

### wäre (sein im Konjunktiv II)

| Person | wäre |
|--------|------|
| ich | **wäre** |
| du | **wärst** / wärest |
| er/sie/es | **wäre** |
| wir | **wären** |
| ihr | **wärt** / wäret |
| sie/Sie | **wären** |

### hätte (haben im Konjunktiv II)

| Person | hätte |
|--------|-------|
| ich | **hätte** |
| du | **hättest** |
| er/sie/es | **hätte** |
| wir | **hätten** |
| ihr | **hättet** |
| sie/Sie | **hätten** |

## Konjunktiv II: Wünsche ausdrücken

| Ausdruck | Beispiel |
|---------|---------|
| Ich **würde gern** + Infinitiv | Ich würde gern ins Konzert gehen. |
| Ich **hätte gern** + Nomen | Ich hätte gern mehr Freizeit. |
| Ich **wäre gern** + Adjektiv/Nomen | Ich wäre gern Künstler. |
| lieber | Ich würde **lieber** wandern. |
| am liebsten | Am liebsten würde ich reisen. |

## Konjunktiv II: Vorschläge machen

| Ausdruck | Beispiel |
|---------|---------|
| Du **könntest** ... | Du könntest ins Kino gehen. |
| Man **könnte** ... | Man könnte einen Ausflug machen. |
| Wir **könnten** ... | Wir könnten in die Oper gehen. |
| Sie **sollten** ... | Sie sollten mehr Sport treiben. |

**könnten (Konjunktiv II von können):**

| Person | könnte |
|--------|--------|
| ich | könnte |
| du | könntest |
| er/sie/es | könnte |
| wir | könnten |
| ihr | könntet |
| sie/Sie | könnten |
