# Grammatik – Lektion 12: Reisen

## Adjektivdeklination: Ohne Artikel (No Article)

| | maskulin | feminin | neutral | Plural |
|--|----------|---------|---------|--------|
| Nom. | **schöner** Bauernhof | **schöne** Aussicht | **schönes** Frühstück | **regionale** Produkte |
| Akk. | **schönen** Bauernhof | **schöne** Aussicht | **schönes** Frühstück | **regionale** Produkte |
| Dat. | **schönem** Bauernhof | **schöner** Aussicht | **schönem** Frühstück | **regionalen** Produkten |

**Typische Kontexte (often used in accommodation descriptions):**
- **Leckeres** Frühstück inklusive.
- **Familiäre** Atmosphäre im Haus.
- Zimmer mit **schönem** Blick auf den See.

## Temporale Präpositionen (Time Prepositions)

| Präposition | Kasus | Frage | Beispiel |
|-------------|-------|-------|---------|
| **an** | Dativ | Wann? | **am** Montag, **am** Wochenende |
| **in** | Dativ | Wann? | **im** Juli, **im** Sommer |
| **vor** | Dativ | Wann? | **vor** zwei Tagen, **vor** dem Urlaub |
| **nach** | Dativ | Wann? | **nach** dem Frühstück, **nach** dem Urlaub |
| **seit** | Dativ | Seit wann? | **seit** einer Woche |
| **bis** | Akkusativ | Bis wann? | **bis** Freitag, **bis** zum Ende |
| **für** | Akkusativ | Wie lange? | **für** drei Tage |
| **während** | Genitiv | Während? | **während** des Urlaubs |

## Modale Präposition: ohne + Akkusativ

| Struktur | Beispiel |
|---------|---------|
| **ohne** + Akkusativ (+ zu + Infinitiv) | Er fährt **ohne** einen Plan. |
| | Sie reist **ohne** Gepäck. |
| | Er geht, **ohne** zu bezahlen. |

## Konjunktiv II: Wunsch und Bedingung (Revision)

| Ausdruck | Beispiel |
|---------|---------|
| Ich **würde gern** + Infinitiv | Ich würde gern ans Meer fahren. |
| Ich **wäre gern** … | Ich wäre gern am Strand. |
| Ich **hätte gern** … | Ich hätte gern ein Zimmer mit Meerblick. |
| Wenn ich Zeit **hätte**, … | Wenn ich Zeit hätte, würde ich reisen. |
| Wenn ich Geld **hätte**, … | Wenn ich Geld hätte, würde ich ins Hotel gehen. |

## Satzverbindungen: trotzdem / deshalb / obwohl

| Konjunktion/Adverb | Typ | Beispiel |
|-------------------|-----|---------|
| **obwohl** | Konjunktion (Verb → Ende) | **Obwohl** es regnet, machen wir einen Ausflug. |
| **trotzdem** | Adverb (Pos. 1) | Es regnet. **Trotzdem** machen wir einen Ausflug. |
| **deshalb** | Adverb (Pos. 1) | Es regnet. **Deshalb** bleiben wir drinnen. |
| **weil** | Konjunktion (Verb → Ende) | Wir bleiben drinnen, **weil** es regnet. |
