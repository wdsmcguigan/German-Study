# Grammatik – Lektion 9: Reisen und Erlebnisse

## Konjunktionen: während, nachdem, bevor

These three subordinating conjunctions describe time relationships between events. The conjugated verb goes to the end of the subordinate clause.

### während (while / whereas)

**während** = "while" (simultaneous actions) or "whereas" (contrast)

**Beispiele (Examples):**
- **Während** er schläft, arbeite ich noch.
- **Während** ich kochte, telefonierte er.
- Er hört Musik, **während** er lernt.

> When **während** expresses contrast (whereas), it works like "on the other hand": *Ich liebe Reisen, während er lieber zu Hause bleibt.*

---

### nachdem (after)

**nachdem** = "after" — the action in the **nachdem**-clause happens **first**, then the main clause action.

**Tense sequence with nachdem:**

| Hauptsatz (Zeitform) | Nebensatz mit nachdem |
|----------------------|-----------------------|
| Präsens | Perfekt |
| Präteritum | Plusquamperfekt |

**Beispiele (Examples):**
- **Nachdem** wir gegessen **haben**, gehen wir spazieren. (Präsens + Perfekt)
- **Nachdem** er die Ausbildung **gemacht hatte**, fand er einen Job. (Präteritum + Plusquamperfekt)
- Sie rief an, **nachdem** sie angekommen **war**.

> The **nachdem**-clause always uses a **completed** tense (Perfekt or Plusquamperfekt) because that action is already done when the main clause happens.

---

### bevor (before)

**bevor** = "before" — the main clause action happens before the **bevor**-clause action.

**Beispiele (Examples):**
- **Bevor** ich ins Bett gehe, putze ich die Zähne.
- Er hatte alles vorbereitet, **bevor** die Gäste kamen.
- **Bevor** du fährst, ruf mich bitte an.

> Unlike **nachdem**, **bevor** does not require a tense shift — both clauses can use the same tense.

---

### Summary: Temporal conjunctions

| Konjunktion | Bedeutung | Zeitverhältnis |
|-------------|-----------|----------------|
| **während** | while / whereas | simultaneous |
| **nachdem** | after | Nebensatz = before main clause |
| **bevor** | before | Nebensatz = after main clause |

---

## Konjunktion: als ob

**als ob** = "as if / as though" — used to describe hypothetical comparisons or unreal situations. It introduces a subordinate clause using **Konjunktiv II**.

**Structure:** … als ob + Konjunktiv II (verb at end of clause)

### Konjunktiv II forms

| Verb | Konjunktiv II |
|------|---------------|
| sein | wäre |
| haben | hätte |
| werden | würde |
| wissen | wüsste |
| können | könnte |

**Beispiele (Examples):**
- Er tut so, **als ob** er keine Ahnung **hätte**.
- Sie sieht aus, **als ob** sie krank **wäre**.
- Er spricht, **als ob** er alles **wüsste**.
- Das klingt, **als ob** es ein Problem **gäbe**.

> Shortened form **als** (without **ob**) is also possible — then the verb comes immediately after **als**:
> - Er tut so, **als wäre** er der Chef.
> - Sie sieht aus, **als hätte** sie nicht geschlafen.

> **als ob** always triggers Konjunktiv II because it describes something unreal or merely imagined.
