# Grammatik – Lektion 6: Verben mit "es", Infinitivkonstruktionen und Konjunktionen

## Verben und Ausdrücke mit "es"

Some German verbs and expressions always use **es** as a fixed subject or object. This **es** cannot be replaced by another word.

### Allgemeine Ausdrücke

| Ausdruck | Beispiel |
|---------|---------|
| Es ist (nicht) leicht/schön/schwierig/möglich | **Es ist** (nicht) leicht, das zu verstehen. **Es lohnt sich**. |
| Es gibt ... | **Es gibt** hier viele Möglichkeiten. |

### Tages- und Jahreszeiten, Wetter

| Kategorie | Ausdrücke |
|-----------|----------|
| Tageszeiten | Es ist vier Uhr / Sommer / Nacht / ... |
| Wetter | **Es** regnet. **Es** ist heiß. **Es** sind dreißig Grad. / Wie geht's? |

### Befinden

| Ausdruck | Beispiel |
|---------|---------|
| Wie geht es Ihnen? | Wie geht**'s**? |

**Beispiele (Examples):**
- **Es** ist (nicht) leicht/schön/schwierig/möglich ...
- **Es gibt** hier gute Schulen.
- **Es** regnet/ist heiß/sind dreißig Grad.
- Wie geht **es** Ihnen? Es lohnt **sich**.

## Konjunktionen: um … zu + Infinitiv und damit

Both **um … zu** and **damit** express **purpose** (= in order to). Use **um … zu** when both clauses have the **same subject**; use **damit** when the subjects are **different**.

### um … zu + Infinitiv (same subject)

| Struktur | Beispiel |
|---------|---------|
| Hauptsatz + um + … + zu + Infinitiv | Leon will als Erster beim Laden sein, **um** dort Schuhe **zu kaufen**. |
| | Leon will als Erster beim Laden sein, **um** dort Schuhe **kaufen zu können**. |

### damit (different subjects)

| Struktur | Beispiel |
|---------|---------|
| Hauptsatz + damit + Nebensatz (Verb → Ende) | Leon will als Erster beim Laden sein, **damit** er dort Schuhe kaufen kann. |
| | Leon kauft diese Schuhe, **damit** sein Kunde ausschlafen kann. |

**Beispiele aus dem Buch:**
- Leon will als Erster beim Laden sein, **um** dort Schuhe **zu kaufen**.
- Leon will als Erster beim Laden sein, **damit** er dort Schuhe kaufen kann.
- Leon kauft diese Schuhe, **damit** sein Kunde ausschlafen kann.

> Same subject → **um … zu** + Infinitiv
> Different subjects → **damit** + conjugated verb at end

## Konjunktionen: statt/ohne … zu + Infinitiv

**statt … zu** and **ohne … zu** combine with an infinitive to replace longer subordinate clauses with **anstatt dass / ohne dass**. Used when both clauses share the **same subject**.

| Konjunktion | Bedeutung | Beispiel |
|-------------|-----------|---------|
| **statt** … **zu** + Inf. | instead of (doing) | Man sollte etwas tun, **statt** nur **zu träumen**. |
| **ohne** … **zu** + Inf. | without (doing) | Man kann nichts Neues lernen, **ohne** Fehler **zu machen**. |

**Beispiele aus dem Buch:**
- Man sollte etwas tun, **statt** nur **zu träumen**.
- Man kann nichts Neues lernen, **ohne** Fehler **zu machen**.

**Weitere Beispiele (Examples):**
- Er ging, **ohne** ein Wort **zu sagen**.
- Sie arbeitet, **statt** sich **zu erholen**.
- Er löst das Problem, **ohne** andere um Hilfe **zu bitten**.

> With separable verbs: **aufhören** → **aufzuhören**
> Same subject required for statt/ohne … zu

## Zweiteilige Konjunktionen (Correlative Conjunctions)

These conjunctions work in pairs and connect two parallel elements. They can link nouns, adjectives, or entire clauses.

| Zweiteilige Konjunktion | Bedeutung | Beispiel |
|------------------------|-----------|---------|
| **nicht nur** … **sondern auch** | not only … but also | Das macht ja **nicht nur** Lärm, **sondern** stört **auch** die Nachbarn. |
| **zwar** … **aber** | (it's true that) … but | Ich habe **zwar** Lösungsvorschläge gemacht, **aber** niemand hat zugehört. |
| **entweder** … **oder** | either … or | Wir sollen **entweder** eine leisere Säge verwenden, **oder** ... |
| **nicht nur** … **aber** / **sondern auch** | see above | |

**Beispiele aus dem Buch:**
- Das macht ja **nicht nur** Lärm, **sondern** stört **auch** die Nachbarn. / eine leise Säge verwenden, ...
- Ich habe **zwar** Lösungsvorschläge gemacht, **aber** ...
- Wir sollen **entweder** ... **oder** ...

**Weitere Beispiele (Examples):**
- Er spricht **nicht nur** Deutsch, **sondern auch** Englisch.
- Das ist **zwar** teuer, **aber** es lohnt sich.
- **Entweder** du kommst mit, **oder** ich gehe alleine.

> **nicht nur … sondern auch**: sondern is used (not aber) because the second part ADDS to/corrects the first.
> **zwar … aber**: "zwar" concedes a point, "aber" introduces the contrast.
