# Grammatik – Lektion 6: Bildungswege

## Partizipial-Konstruktionen (Participial Phrases)

Partizip I and II can be used as adjectives before nouns.

### Partizip I (present participle = ongoing action)
**Formation:** Infinitiv + **-d** → used as adjective with endings

| Infinitiv | Partizip I | Beispiel |
|-----------|-----------|---------|
| studieren | studierend- | ein **studierender** Ausländer |
| wachsen | wachsend- | die **wachsende** Wirtschaft |
| steigen | steigend- | **steigende** Kosten |
| lernen | lernend- | **lernende** Schüler |

### Partizip II (past participle = completed/passive action)
| Partizip II | Beispiel |
|-------------|---------|
| abgeschlossen | ein **abgeschlossenes** Studium |
| empfohlen | das **empfohlene** Buch |
| qualifiziert | ein **qualifizierter** Bewerber |

## Indirekte Rede (Reported Speech)

Used to report what someone else said.

**Mit Konjunktiv I:**

| Person | sein | haben |
|--------|------|-------|
| er/sie | **sei** | **habe** |
| sie (Pl.) | **seien** | **haben** |

**Beispiele:**
- Er sagt, er **sei** krank. (Er sagt: „Ich bin krank.")
- Sie sagt, sie **habe** bestanden. (Sie sagt: „Ich habe bestanden.")
- Sie behaupten, sie **seien** qualifiziert.

**Wenn Konjunktiv I = Indikativ → Konjunktiv II verwenden:**
- Sie sagten, sie **hätten** keine Zeit. (haben → hätten, weil „haben" = Indikativ)

**Mit dass (direkter Nebensatz):**
- Er sagt, **dass** er bestanden **habe**.
- Sie erklärte, **dass** sie sich beworben **habe**.

## Fragepronomen: Wer / Was in Relativsätzen

| Bezug | Relativpronomen | Beispiel |
|-------|----------------|---------|
| eine Person (unbestimmt) | **wer** | **Wer** fleißig lernt, besteht die Prüfung. |
| eine Sache (unbestimmt) | **was** | **Was** er gelernt hat, hat er nicht vergessen. |
| Superlativ | **was** | Das Beste, **was** ich je gelernt habe. |
| alles / nichts / etwas | **was** | Alles, **was** er weiß, ist wichtig. |
