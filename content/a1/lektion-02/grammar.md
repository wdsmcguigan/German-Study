# Grammatik – Lektion 2: Meine Familie

## Verb: haben (to have)

| Person | Form |
|--------|------|
| ich | **habe** |
| du | **hast** |
| er / sie / es | **hat** |
| wir | **haben** |
| ihr | **habt** |
| sie / Sie | **haben** |

> *Ich **habe** einen Bruder. · Sie **hat** zwei Kinder.*

## Possessivartikel (Possessive Articles)

| Pronomen | maskulin | feminin | neutral | Plural |
|----------|----------|---------|---------|--------|
| ich → | **mein** | **meine** | **mein** | **meine** |
| du → | **dein** | **deine** | **dein** | **deine** |
| er → | **sein** | **seine** | **sein** | **seine** |
| sie → | **ihr** | **ihre** | **ihr** | **ihre** |
| wir → | **unser** | **unsere** | **unser** | **unsere** |
| Sie → | **Ihr** | **Ihre** | **Ihr** | **Ihre** |

**Beispiele:**
- Das ist **mein** Vater. (m)
- Das ist **meine** Mutter. (f)
- Das ist **mein** Kind. (n)
- Das sind **meine** Kinder. (Pl)

## Nomen: Plural (Noun Plurals)

German plurals are unpredictable — learn each noun with its plural.

| Singular | Plural | Typ |
|----------|--------|-----|
| der Bruder | die Brüder | Umlaut |
| die Schwester | die Schwestern | -n |
| der Vater | die Väter | Umlaut |
| die Mutter | die Mütter | Umlaut |
| das Kind | die Kinder | -er |
| der Mann | die Männer | Umlaut + -er |
| die Frau | die Frauen | -en |
| das Jahr | die Jahre | -e |

## Verb: arbeiten / wohnen (regular -en verbs)

| Person | wohnen | arbeiten |
|--------|--------|---------|
| ich | wohne | arbeite |
| du | wohnst | arbeitest |
| er/sie/es | wohnt | arbeitet |
| wir | wohnen | arbeiten |
| ihr | wohnt | arbeitet |
| sie/Sie | wohnen | arbeiten |

> Verbs ending in **-t, -d**: add extra **e** before **-st / -t** endings (arbeit**e**st, arbeit**e**t)

## Zahlen: Telefonnummern lesen

Telefonnummern werden in Gruppen gelesen:
- 0163 / 235 / 61 / 47
- Null-eins-sechs-drei / zweihundertfünfunddreißig / einundsechzig / siebenundvierzig

## Himmelsrichtungen + in/im

| Richtung | Beispiel |
|----------|---------|
| der Norden | Hamburg ist **im Norden**. |
| der Süden | München ist **im Süden**. |
| der Osten | Leipzig ist **im Osten**. |
| der Westen | Frankfurt ist **im Westen**. |

> **im** = in + dem (contraction)
