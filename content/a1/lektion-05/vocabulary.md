## Uhrzeit (Clock Time)

| Deutsch | Englisch |
|---------|----------|
| die Uhr | clock / o'clock |
| die Stunde | hour |
| die Minute | minute |
| das Viertel | quarter |
| halb | half |
| vor | before |
| nach | after |
| Wieviel Uhr ist es? | What time is it? |
| Es ist ... Uhr. | It is ... o'clock. |
| kurz vor / nach | shortly before / after |

**Beispiele:**
- 9:00 → Es ist neun Uhr.
- 9:15 → Es ist Viertel nach neun.
- 9:30 → Es ist halb zehn.
- 9:45 → Es ist Viertel vor zehn.
- 9:05 → Es ist fünf nach neun.
