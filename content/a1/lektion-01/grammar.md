# Grammatik – Lektion 1: Ich heiße...

## Verb: sein (to be)

| Person | Form |
|--------|------|
| ich | **bin** |
| du | **bist** |
| er / sie / es | **ist** |
| wir | **sind** |
| ihr | **seid** |
| sie / Sie | **sind** |

> *Ich **bin** Lili. · Er **ist** Walter. · Wir **sind** aus Deutschland.*

## Verb: heißen (to be called)

| Person | Form |
|--------|------|
| ich | **heiße** |
| du | **heißt** |
| er / sie / es | **heißt** |
| wir | **heißen** |
| ihr | **heißt** |
| sie / Sie | **heißen** |

## Verb: kommen / sprechen (regular pattern)

| Person | kommen | sprechen |
|--------|--------|---------|
| ich | komme | spreche |
| du | kommst | sprichst |
| er/sie/es | kommt | spricht |
| wir | kommen | sprechen |
| ihr | kommt | sprecht |
| sie/Sie | kommen | sprechen |

> *sprechen* hat Vokalwechsel: e → i (du sprich**st**, er sprich**t*)

## Aussagesatz – Wortstellung (Word Order: Statement)

Der Verb steht immer an **Position 2**.

| Position 1 | Position 2 (Verb) | Rest |
|-----------|-------------------|------|
| Ich | **komme** | aus Deutschland. |
| Mein Name | **ist** | Walter Baumann. |

## W-Frage (Question with Question Word)

| Position 1 (W-Wort) | Position 2 (Verb) | Rest |
|--------------------|-------------------|------|
| Wie | **heißen** | Sie? |
| Woher | **kommen** | Sie? |
| Wo | **wohnen** | Sie? |
| Wer | **ist** | das? |

## Ja/Nein-Frage

Der Verb kommt an **Position 1**.

| Position 1 (Verb) | Rest |
|-------------------|------|
| **Sprechen** | Sie Deutsch? |
| **Kommen** | Sie aus Polen? |

> Antwort: *Ja.* / *Nein.*

## Länder → Adjektive / Sprachen

| Land | Adjektiv | Sprache |
|------|----------|---------|
| Deutschland | deutsch | Deutsch |
| Österreich | österreichisch | Deutsch |
| die Schweiz | schweizerisch | Deutsch / Französisch |
| Polen | polnisch | Polnisch |
| die Türkei | türkisch | Türkisch |
| Spanien | spanisch | Spanisch |
| Italien | italienisch | Italienisch |
| Griechenland | griechisch | Griechisch |

> **Aus + Land (Dativ):** Ich komme **aus** Deutschland. / Ich komme **aus der** Schweiz. / Ich komme **aus dem** Iran.
