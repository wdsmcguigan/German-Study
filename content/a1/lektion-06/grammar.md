# Grammatik – Lektion 6: Freizeit

## Modalverb: wollen (to want to)

| Person | Form |
|--------|------|
| ich | **will** |
| du | **willst** |
| er / sie / es | **will** |
| wir | **wollen** |
| ihr | **wollt** |
| sie / Sie | **wollen** |

**Satzstruktur:** Modal an Position 2, Infinitiv ans Ende.

| Pos. 2 (Modal) | Ende (Infinitiv) |
|----------------|-----------------|
| Ich **will** | wandern. |
| Sie **will** | nicht fotografieren. |
| Wir **wollen** | Freunde treffen. |

## Modalverben im Vergleich (Modal Verbs Compared)

| | möchten | können | wollen |
|--|---------|--------|--------|
| **Bedeutung** | would like to | can / am able to | want to |
| **ich** | möchte | kann | will |
| **du** | möchtest | kannst | willst |
| **er/sie** | möchte | kann | will |
| **wir** | möchten | können | wollen |

**Beispiele:**
- Ich **möchte** ein Eis essen. (polite wish)
- Ich **kann** gut kochen. (ability)
- Ich **will** heute wandern. (strong intention)

## Ja/Nein-Frage: ja – nein – doch

| Frage | Antwort |
|-------|---------|
| Machst du Sport? | **Ja**, ich mache Sport. |
| Machst du Sport? | **Nein**, ich mache keinen Sport. |
| Machst du **keinen** Sport? | **Doch!** (= Yes I do!) |
| Hast du **keine** Zeit? | **Doch**, ich habe Zeit. |

> **doch** = yes (in response to a negative question / contradiction)

## Akkusativ: Verben mit direktem Objekt

Häufige Verben mit Akkusativ:
- kaufen, nehmen, essen, trinken, lesen, sehen, hören, haben, mögen, brauchen, möchten, wollen

| Satz | Fall |
|------|------|
| Ich kaufe **einen** Salat. | Akk. mask. |
| Ich trinke **die** Cola. | Akk. fem. |
| Ich lese **das** Buch. | Akk. neutr. |
| Ich möchte **keinen** Kuchen. | Akk. mask. neg. |

## Wetter beschreiben (Describing Weather)

| Struktur | Beispiel |
|---------|---------|
| Es ist + Adjektiv | Es ist sonnig / warm / kalt. |
| Es + Verb | Es regnet. / Es schneit. / Es scheint. |
| Es gibt + Akkusativ | Es gibt viele Wolken. / Es gibt Wind. |
| Die Temperatur + ist | Die Temperatur ist 20 Grad. |
| Es sind + Grad | Es sind 25 Grad. |
| Es ist minus + Zahl | Es ist minus 5 Grad. |

## Jahreszeiten + im (Seasons with im)

| Jahreszeit | Präposition | Beispiel |
|-----------|------------|---------|
| der Frühling | **im** Frühling | Im Frühling blühen Blumen. |
| der Sommer | **im** Sommer | Im Sommer ist es warm. |
| der Herbst | **im** Herbst | Im Herbst regnet es oft. |
| der Winter | **im** Winter | Im Winter schneit es. |
